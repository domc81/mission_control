# 2025-02-09 - Mission Control Build Error & Lessons

## The Error
- Build failed: `Module not found: Error: Can't resolve './App' in '/root/.openclaw/workspace-cestra/ui/src'`
- Root cause: Import path `./App` in `index.tsx` should work for CRA, but I didn't actually run `npm run build` to verify

## Key Lessons
1. **Test before claiming completion** - I wrote "MISSION CONTROL REFACTOR COMPLETE" without running `npm run build` to verify. Never again.
2. **Build verification is mandatory** - Any code change affecting the build must be tested with `npm run build`
3. **Cache issues** - Clear with `rm -rf node_modules/.cache` when weird errors occur

## Correct Build/Deploy Process
```bash
cd /root/.openclaw/workspace-cestra/ui
rm -rf node_modules/.cache
npm run build

# Serve locally for testing
cd build
python3 -m http.server 8080
```
